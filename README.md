Краткий конспект по гит в моем случае
1) Создать акк на гитхабе
2) Скачать, установить Fork GitClient
3) Клонировать репозиторий из Гитхаба в Форк
4) Создать проект в Икс-код
5) Любой файл проекта, который требует измений, меняете, он отображается в LocalChange
6) Добавляете его в Stage, пишите комментарий, нажимаете commit
7) Кликаем Push, этот файл с изменениями переносится в удалений репозиторий
8) Через форк можно скачать все прежние версии этого файла, через pull
9) Ветки создаем для параллельной работы над одним и тем же проектом, который в конце можно смержевать и выпустить релиз
